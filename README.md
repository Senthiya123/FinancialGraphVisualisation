# FinancialGraphVisualisation
This is a wealth planning modelling tool for Insight Wealth Planning, an Australian financial advisory brand. It helps advisers and clients compare current vs proposed financial strategies and see the impact over time.
 
Tech Stack
Language -	TypeScript 5.7
Framework	- React 19
Build -	Vite 6
UI -	Chakra UI 2, MUI 7, MUI X Charts
Drag & Drop -	react-dnd (HTML5 backend)
Icons -	react-icons
Animations - framer-motion
